# Candidate Form
Candidate Form
(Тестовое задание)

## Stack

- React
- Formik
- Sass modules

## Structure

- `/src/components` - components folder
- `/src/common` - reusable components

## Commands
### yarn
```bash
# Install modules
yarn install 
# Run app in dev mode
yarn start
# Create production build
yarn build
```

### npm
```bash 
# Install modules
npm install 
# Run app in dev mode
npm start
# Create production build
npm build
```
